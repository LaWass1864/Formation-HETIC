Introduction à JavaScript

Les bases pour débutants

C'est quoi JavaScript ?

  • Rend les pages web interactives
  • Réagit aux actions de l'utilisateur
  • Change le contenu sans recharger la page

Où placer le code ?

Option 1 : Dans le HTML

<body> <!-- Votre contenu HTML --> <script> console.log("Hello"); </script> </body>

Où placer le code ?

Option 2 : Fichier séparé (recommandé)

<body> <!-- Votre contenu HTML --> <script src="script.js"></script> </body>

Créer un fichier script.js à côté du HTML

La console

Votre meilleur ami pour débugger

  • Ouvrir : F12 ou Clic droit → Inspecter
  • Afficher des messages : console.log()
console.log("Bonjour !");

Données du profil

Variable

  • Peut être modifiée
  • Mot-clé : let
let username = "netflixfr";

Constante

  • Ne peut PAS être modifiée
  • Mot-clé : const
  • Exemple : prénom, date de naissance
const prenom = "Alice"; const verified = true;

Types de données

Type Exemple
String "netflixfr"
Number 10500000
Boolean true / false

Exemple complet

let username = "netflixfr"; let followers = 10500000; const verified = true;

Manipuler le HTML

DOM = Document Object Model

  • Sélectionner un élément
  • Modifier son contenu
  • Modifier son style

ID en HTML

Identifiant unique d'un élément

<h1 id="username">netflixfr</h1>
  • Classe : peut être utilisée plusieurs fois
  • ID : unique, une seule fois par page

Sélectionner un élément

querySelector = fonction pour cibler un élément HTML

Sélectionner par ID :

let titre = document.querySelector("#username");

Le # sert à cibler un ID

Sélectionner par classe :

let titre = document.querySelector(".card");

Le . sert à cibler une classe

Modifier le texte

titre.textContent = "netflixfr";

Change le texte de l'élément

Modifier le style

titre.style.color = "blue"; titre.style.fontSize = "40px";

Réagir aux clics

let bouton = document.querySelector("#monBouton"); bouton.addEventListener("click", function() { alert("Le bouton a été cliqué !"); });

Conditions

Prendre des décisions

  • Si une condition est vraie
  • Alors faire une action
  • Sinon faire autre chose

if / else

if (followers > 1000000) { badge.textContent = "✅ Vérifié"; badge.style.color = "green"; } else { badge.textContent = "Non vérifié"; badge.style.color = "gray"; }

Exemple complet

// Variables let followers = 10500000; // Sélectionner l'élément let bouton = document.querySelector("#verifier"); // Événement au clic bouton.addEventListener("click", function() { if (followers > 1000000) { alert("✅ Compte vérifié !"); } });

Récapitulatif

  • Variables : let
  • Constantes : const
  • Sélectionner : querySelector()
  • Modifier : textContent, style
  • Événement : addEventListener()
  • Condition : if / else
